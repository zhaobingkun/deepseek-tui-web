改名说明：DeepSeek TUI 现在叫 CodeWhale

上游项目已经正式改名。新安装应优先使用 npm install -g codewhale，启动命令也应优先使用 codewhale。

旧的 deepseek / deepseek-tui 目前还作为兼容 shim 保留，但上游已说明这只是过渡方案，并计划在 v0.9.0 移除。
