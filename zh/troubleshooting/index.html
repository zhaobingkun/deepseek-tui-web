排错总览

DeepSeek TUI 的常见问题，大多还是安装、路径和配置基础问题

很多“工具坏了”的直觉，其实最后都落在命令没进 path、provider 没选对、API Key 没生效这些基础点上。

优先级建议

先查命令能不能调用，再查配置和 provider，最后再怀疑更复杂的问题。

大多数排错都不需要从最复杂的方向开始。

Command Not Found

command not found 修复

安装后最常见的第一类错误，通常和 shell path 或安装路径有关。

打开修复页
Homebrew

Homebrew command not found

如果 brew 说装好了，但终端还是找不到命令，这页更对应。

打开 Homebrew 修复
Provider

Provider 排错

命令存在但行为不对时，provider 线通常比安装线更重要。

打开 provider 排错
Binaries

Release binaries

如果包管理器本身就是阻力，二进制安装这条线更值得单独看。

打开 release binaries
API Key

API Key 与 provider

如果命令存在但无法使用，配置线通常比安装线更关键。
