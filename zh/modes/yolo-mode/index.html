Yolo Mode

Yolo mode 适合更直接的推进方式，但前提是你能接受更高的操作风险

很多人搜索 yolo mode，是因为他们想知道：到底值不值得为了速度换取更少的中间确认。

直接答案

如果你想减少停顿、加快推进速度，yolo mode 会更贴近你的工作方式。

但它并不适合每一种环境，尤其是高风险命令场景。

为什么这一页值得单独做成中文

这个主题本身已经开始从英文 README 搜索，转向更具体的中文安装、配置、模式和排错问题，所以单独保留一个中文入口是有价值的。
