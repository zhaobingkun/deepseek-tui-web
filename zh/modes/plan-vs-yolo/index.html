模式对比

Plan mode vs yolo mode 是 DeepSeek TUI 里最值得单独做成决策页的一组模式对比

有些用户并不想先分别读两页，而是想直接看一页：到底哪个模式更适合当前任务、风险和节奏。

直接答案

如果你更在意审查和可控性，就偏向 plan mode；如果你更在意推进速度，就偏向 yolo mode。

真正的选择标准，是你愿意让代理直接行动到什么程度。

这页到底解决什么问题

这页适合已经知道两个 mode 名字，但还不知道哪一个适合真实终端工作流的人。决定因素通常是信任程度、任务风险和你对中间确认的容忍度。

先比什么

  • 你想要多少前置规划
  • 你能接受多少直接执行风险
  • 当前任务是探索型，还是高风险型
