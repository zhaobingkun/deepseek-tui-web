配置总览

DeepSeek TUI 的配置问题，通常比安装问题更常见

很多用户已经把命令装上了，但真正开始用时才发现 provider、凭据加载或配置文件路径才是第一个卡点。

直接建议

如果命令已经能跑，但结果不对，先查 provider 和 API Key，不要一上来怀疑高级功能。

首次使用的大部分失败，都来自基础配置而不是复杂的代理逻辑。

按问题类型选择配置路径
