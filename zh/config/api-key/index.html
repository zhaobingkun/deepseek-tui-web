API Key

DeepSeek TUI 装好以后，API Key 与 provider 配置往往是第一个真正的检查点

如果命令已经装上了，但工具还不能真正工作，最可能的失败点通常不是高级 agent bug，而是 provider 选择、凭据加载或配置文件路径。

直接答案

先把 provider 和 API Key 配对正确，再去排查更复杂的问题。

首次运行的大部分失败，本质上都是基础配置问题。
