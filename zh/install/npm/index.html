npm 安装

如果你本来就用 Node 管理终端工具，npm 往往是最省事的 DeepSeek TUI 安装方式

对于已经习惯用 Node 全局安装 CLI 工具的开发者来说，npm 路线的学习成本最低，更新方式也最熟悉。

命令

npm install -g deepseek-tui

如果你追求快速装好并开始用，npm 通常是最直接的一条路。

什么时候优先选 npm

如果你的终端环境已经长期依赖 Node 全局 CLI，继续用 npm 会比临时引入 cargo 更简单。

安装后先检查什么

  • 确认命令已经进入 shell path。
  • 确认 API Key 和 provider 已经准备好。
  • 确认你知道配置文件放在哪里，再开始修改设置。
