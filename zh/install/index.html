安装总览

安装 DeepSeek TUI 前，先知道它已经改名为 CodeWhale，再选包管理路径

这个安装总页保留旧搜索词，但新的安装与启动命令应优先使用 CodeWhale。真正要先想清楚的，不只是命令本身，而是你当前 CLI 环境更适合哪条更新和维护路径。

快速结论

新的 npm 安装应优先使用 npm install -g codewhale。如果你长期在 Rust 或 Homebrew 生态里，再看对应路线。

系统级安装、Homebrew 和预编译二进制仍然有价值，但不要把旧命令和新命令混在一起。

改名说明：产品名现在是 CodeWhale

上游已经把 DeepSeek TUI 改名为 CodeWhale。新的安装命令应优先使用 codewhale 相关名称，尤其是 npm install -g codewhale。

旧的 deepseek / deepseek-tui 目前还作为兼容 shim 存在，但上游已明确这只是过渡方案，并计划在 v0.9.0 移除。

当前命令
npm：`npm install -g codewhale`更新：`codewhale update`官方安装文档
