文档总览

DeepSeek TUI 的文档值得单独成一个 hub，因为仓库文档目录和搜索意图并不是一回事

这个 hub 把上游 docs 目录重新整理成更适合搜索和浏览的结构，覆盖安装、配置、MCP、运行时、架构和维护者相关主题。

最佳用法

如果你已经知道自己想看哪类主题，这个 hub 会比直接翻仓库更快。

先看核心文档解决上手问题，再决定是否进入架构和 runbook。
