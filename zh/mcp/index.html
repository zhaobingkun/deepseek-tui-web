MCP 总览

DeepSeek TUI 的 MCP，不是边角功能，而是很多高级工作流的关键入口

安装和基础配置解决之后，很多搜索会自然转向 MCP，因为它影响工具访问、上下文扩展和代理能力边界。

一句话理解

如果你在意工具接入、外部上下文和扩展能力，MCP 就值得单独学习。

它通常属于进阶阶段，但不等于只有高级用户才会搜。
