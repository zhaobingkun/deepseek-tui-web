关于这个独立的 DeepSeek TUI 与 CodeWhale 中文指南

DeepSeek TUI 中文指南是一个独立的安装、配置、迁移和排错站点，面向仍然搜索 DeepSeek TUI，但实际需要理解当前 CodeWhale 名称、安装命令、配置路径、模式、MCP 和常见错误的用户。

本站不是 CodeWhale 官方站，也不是上游仓库镜像。它的目标是把分散在 release note、README、命令说明和排错经验里的信息整理成更容易执行的决策路径，帮助用户先判断自己该看哪一页，再进入更深的技术细节。

站点范围
  • 安装与更新流程
  • API Key、Provider 与配置路径
  • DeepSeek TUI 改名到 CodeWhale 的迁移说明
  • 模式、MCP、Skills 与常见错误排查

内容如何编辑

本站页面围绕具体任务来写：如何安装当前包、如何确认真正生效的二进制、如何区分 shell、provider、配置文件和上游行为问题，以及一个用户下一步应该打开哪个页面。需要核对来源时，页面会保留官方或上游链接，但本站的价值不在复制原文，而在给出更清楚的操作顺序。

当上游名称、命令或安装路径变化时，优先更新首页、安装页、provider setup、command not found 排错、对比页和改名说明页。旧文档镜像页只作为参考，不作为本站主要价值。

本站重点解决什么

  • 把 DeepSeek TUI 旧搜索词和 CodeWhale 当前名称分开解释。
  • 帮助用户选择 npm、cargo、Homebrew、Windows 或其他安装路径。
  • 用常见失败分支来解释问题，而不是只堆命令。
  • 把用户从“我不知道哪里坏了”引导到具体安装、配置、模式或排错页面。

修正与更新

如果你发现命令、包名、provider 行为或迁移说明已经过时，请通过联系页反馈。最有用的信息包括页面 URL、出错段落、你测试的平台、当前输出或可核对的来源。
