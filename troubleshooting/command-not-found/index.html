Fix

DeepSeek TUI command not found usually means the install succeeded but the shell path did not

This problem appears often because users focus on the install command and forget to verify where the package manager placed the executable.

First Checks

Confirm the install route, then confirm the executable is on your shell path.

If npm or cargo installed correctly, the command is often present but invisible to the current shell environment.
