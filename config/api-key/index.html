API Key

DeepSeek TUI API key setup is usually the first real config checkpoint after install

If the command is installed but the tool still is not useful, the next likely failure point is provider selection, credential loading, or config file placement.

Direct Answer

Get the provider and API key right before you troubleshoot anything more exotic.

Most first-run failures are basic configuration issues, not advanced agent bugs.
